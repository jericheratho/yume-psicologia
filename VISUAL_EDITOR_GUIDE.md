# Manus Visual Editor — Edit Your Website Without Coding

The **Manus Visual Editor** lets you edit your website directly without writing any code. You can change text, images, colors, and layout in real-time.

---

## 🎯 What You Can Edit

✅ **Text Content** — Headlines, descriptions, buttons, labels  
✅ **Images** — Replace photos and graphics  
✅ **Colors** — Change brand colors, backgrounds, accents  
✅ **Layout** — Adjust spacing, padding, alignment  
✅ **Buttons & Links** — Edit CTA buttons, navigation links  
✅ **Sections** — Show/hide or reorder page sections  

---

## 📍 How to Access the Visual Editor

### Step 1: Open Manus Management UI

1. Go to your Manus project dashboard
2. Look for the **"Preview"** panel on the right side
3. You should see your website preview

### Step 2: Enter Edit Mode

1. In the Preview panel, look for an **"Edit"** button or pencil icon
2. Click it to enter visual editing mode
3. Your website is now in edit mode (you'll see selection highlights)

---

## ✏️ Editing Text

### To Edit Any Text:

1. **Click on the text** you want to edit (e.g., a heading, description, button label)
2. The text will be highlighted with a blue border
3. A text editor will appear on the right side
4. **Type your new text** in the editor
5. Changes appear instantly on the preview
6. Click outside or press Enter to confirm

### Example: Edit Hero Heading

1. Click on "Sonhos, acolhimento & muito afeto" (the main heading)
2. The text editor appears
3. Type your new heading
4. See it update in real-time

---

## 🖼️ Changing Images

### To Replace an Image:

1. **Click on the image** you want to replace
2. The image will be highlighted with a blue border
3. An image editor panel appears on the right
4. Click **"Upload image"** or **"Choose from library"**
5. Select your new image
6. The image updates instantly

### Supported Image Formats:
- JPG / JPEG
- PNG
- WebP
- GIF

### Tips:
- Use high-quality images (at least 1200px wide for hero images)
- Compress images before uploading (smaller file size = faster loading)
- Use consistent image styles for professional look

---

## 🎨 Changing Colors

### To Change a Color:

1. **Click on an element** with the color you want to change (button, background, text, etc.)
2. Look for the **"Color"** option in the right panel
3. Click the color picker
4. Choose your new color from the palette or enter a hex code
5. The color updates instantly

### Example: Change Button Color

1. Click on a "Agendar consulta" button
2. Find the "Background color" option
3. Click the color picker
4. Select a new color
5. All buttons with that style update instantly

### Brand Colors (Sage Green):

- Secondary: `#6B9B6B` (dark sage)
- Accent: `#C4896F` (terracotta)
- Background: `#F5F0E8` (cream)
- Text: `#2C2A26` (deep ink)

## 📐 Adjusting Layout & Spacing

### To Change Spacing:

1. **Click on the element** you want to adjust
2. Look for **"Padding"**, **"Margin"**, or **"Spacing"** options
3. Adjust the values (in pixels)
4. See changes in real-time

### To Change Alignment:

1. **Click on the element**
2. Look for **"Alignment"** or **"Text align"** option
3. Choose: Left, Center, Right, or Justify
4. Changes apply instantly

### To Resize Elements:

1. **Click on the element**
2. Look for **"Width"** or **"Height"** options
3. Adjust the values
4. See the element resize in real-time

## 💾 Saving Your Changes

### Auto-Save

The Manus Visual Editor **auto-saves** your changes as you make them. You don't need to manually save!

### Creating Checkpoints

After making significant changes:

1. Go to the **"Checkpoint"** section in Manus
2. Click **"Save checkpoint"**
3. Add a description (e.g., "Updated team info and colors")
4. Your changes are backed up
5. You can rollback to previous versions if needed

---

## 🔄 Rollback Changes

If you make a mistake:

1. Go to **"Checkpoints"** in Manus
2. Find the previous version you want to restore
3. Click **"Rollback"**
4. Your site reverts to that version instantly

## 🎨 Color Editing Tips

### Change All Buttons to a New Color

1. Click on one button
2. Change its color
3. If you're using a consistent color system, all similar buttons update
4. If not, you may need to edit each button individually

### Create a Color Scheme

1. Decide on your color palette (e.g., sage green + cream + dark ink)
2. Use the same colors consistently across the site
3. This creates a professional, cohesive look

### Test Color Contrast

1. Make sure text is readable on background colors
2. Dark text on light backgrounds
3. Light text on dark backgrounds
4. Avoid low-contrast combinations

## 📸 Image Editing Tips

### Best Image Sizes

| Section | Recommended Size |
|---------|-----------------|
| Hero image | 1920 x 1080 px |
| Service cards | 400 x 300 px |
| Team photos | 300 x 300 px |
| Section backgrounds | 1920 x 600 px |

### Image Optimization

1. Compress images before uploading (use TinyPNG or similar)
2. Use WebP format for faster loading
3. Keep file sizes under 500 KB
4. Use consistent image styles

### Image Alignment

1. Click on the image
2. Look for "Object fit" option
3. Choose: Cover, Contain, Fill, or Stretch
4. Adjust alignment as needed

## 🆘 Troubleshooting

### Changes Not Appearing

**Problem:** You edited text but it's not showing

**Solution:**
1. Click outside the element to confirm changes
2. Refresh the preview
3. Check that you're editing the right element

### Image Not Uploading

**Problem:** Image upload fails

**Solution:**
1. Check file size (should be under 5 MB)
2. Check file format (JPG, PNG, WebP, GIF)
3. Try a different image
4. Check your internet connection

### Color Not Changing

**Problem:** Color picker not working

**Solution:**
1. Try clicking the color picker again
2. Enter hex code manually (e.g., #8FBF8F)
3. Refresh the page
4. Try a different element

### Layout Looks Wrong

**Problem:** Elements are misaligned or overlapping

**Solution:**
1. Check padding and margin values
2. Adjust spacing to fix alignment
3. Use the alignment options (left, center, right)
4. Rollback to previous version if needed

### After Editing:

4. Changes appear on your domain

If you're using Netlify:
1. Changes are saved in Manus

4. Your site updates live
